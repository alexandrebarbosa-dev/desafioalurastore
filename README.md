# desafioalurastore
Neste desafio, ajudo o Senhor João a decidir qual loja da sua rede Alura Store vender para iniciar um novo empreendimento. Para isso, analisei os dados de vendas, desempenho e avaliações das 4 lojas fictícias da Alura Store, com o objetivo de identificar a loja com menor eficiência e apresentar uma recomendação final baseada nos dados.
